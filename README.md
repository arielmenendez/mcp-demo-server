# My Advanced Tool Catalog

A simple MCP server built with FastMCP.

## Requirements

-   Python 3.10+
-   Node.js (to run the MCP Inspector)

## Installation

Clone the repository and install dependencies:

``` bash
pip install -r requirements.txt
```

## Running with MCP Inspector

You can run and test the server directly using the MCP Inspector:

``` bash
npx @modelcontextprotocol/inspector
```

Then connect the inspector to your MCP server as instructed.

## Available Tools

-   `search_news`
-   `get_user_profile`
-   `divide`
